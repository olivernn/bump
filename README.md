# Bump

A simple tool for managing project version files.

## Usage

Usage:
    bump init [options]
    bump major [options]
    bump minor [options]
    bump patch [options]
    bump build <build> [options]
    bump pre <pre> [options]
    bump [options]

Options:
    -h, --help           Show this screen
    -f, --file=<file>    Specify the version file path
